# ❄️ aurora-winter-drop-ui

A futuristic winter fashion ecommerce UI built using HTML & CSS.

This project was created while practicing modern frontend layout techniques including CSS Grid, Flexbox, positioning, and ecommerce product showcase structuring.

---

## 🚀 Features

- Futuristic Winter Collection UI
- Modern Ecommerce Layout
- Product Showcase Grid
- Premium Dark Theme Design
- Interactive Product Cards
- Gradient & Glassmorphism Inspired Styling
- Structured Navigation & Footer
- Smooth Layout Composition

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- CSS Grid
- Flexbox
- Remix Icons
- Font Awesome

---

## 📚 Concepts Practiced

- display: grid
- display: flex
- position: relative
- position: absolute
- UI Structuring
- Product Card Layouts
- Grid Based Composition
- Alignment & Spacing
- Modern Ecommerce Design
